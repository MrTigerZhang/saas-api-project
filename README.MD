# SaaS平台 - 公共依赖模块

该模块提供平台级的公共依赖，供各业务服务共享，旨在实现接口标准化、领域模型统一、事件解耦，支持服务间的高效协同。

创建模块后，打包并deploy到maven仓库。依赖它的模块刷新依赖即可。

## 包含内容

- API接口定义（HTTP、Dubbo）
- 领域模型（Domain）
- 枚举与常量
- 消息事件（Event Bus）
- 通用异常与错误码规范

## 使用场景

- 微服务间接口共享
- Dubbo远程调用
- 跨服务事件驱动
- 领域模型复用
